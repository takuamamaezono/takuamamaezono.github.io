Last updated: March 11, 2026

Overview

PowerArQ Marketing Tool is an internal analytics tool that connects to TikTok to retrieve account performance data.

Data Collection

This application collects the following data from TikTok with user authorization:

  • Basic profile information (username, display name)
  • Account statistics (follower count, video count)
  • Video performance metrics (views, likes, comments, shares)

Data Usage

All data is stored in a private Google Spreadsheet and used solely for internal marketing analytics. Data is not shared with third parties.

Data Retention

Users can disconnect their TikTok account and delete all stored data at any time.

Contact

For questions, contact the app administrator.
